# logic-gates-sim

A minimal browser-based logic-gates drawing and simulation project written in TypeScript.

## Quick start

- Open the app in a browser by opening `index.html` in the project root.
- Or serve the project with a static server (recommended during development):

```bash
npm install
# start a simple server (example)
npx serve .
```

## What’s inside

- `src/` — TypeScript sources (Board, tools, UI helpers).
- `index.html` — App entry point.
- `public/` — Static assets.
